## Veterinary Mananger 

---

API REST para la gestión de mascotas y citas médicas de una veterinaria.

### ¿Qué hace este sitema?

Veterinary Mananger es un sistema de gestion para veterinarias, donde ayuda a facilitar la gestion de citas, clientes, y consultas.

### Tecnologias que se utilizaran:

- Java21
- Springbot
- PostgreeSQL
- OpenAPI
- SpringWeb 
- Graddle
- MapStruct
